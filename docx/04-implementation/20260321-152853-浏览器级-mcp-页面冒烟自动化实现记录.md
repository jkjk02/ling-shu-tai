## 核心内容

1. 依赖与配置：
   - 安装 `@playwright/test`。
   - 新增 `playwright.smoke.config.ts`，使用真实后端 `uvicorn` 和 Vite 开发服务器作为 `webServer`。
   - 配置优先复用本机缓存 Chromium；若缓存路径不存在，则退回 Playwright 默认浏览器解析。
2. 冒烟用例：
   - 新增 `tests/browser/mcp-smoke.spec.ts`。
   - 覆盖 `Dashboard`、`Skills`、`Agents`、`Workflows` 页面导航与关键区块可见性。
   - 覆盖 `MCPs` 页面 discovered MCP 状态可见性、详情状态切换和“新建 MCP”对话框基础交互。
3. 最小前端标记增强：
   - `src/layouts/AppLayout.vue` 补导航 `data-testid`。
   - `src/pages/McpsPage.vue` 补“新建 MCP”按钮 `data-testid`。
4. 验证链接入：
   - `package.json` 新增 `test:browser`。
   - `verify` 现在顺序执行 `check:backend`、`test:backend`、`build`、`test:browser`。
5. 文档与 CI：
   - `README.md` 补浏览器冒烟运行方式与覆盖边界。
   - `.github/workflows/ci.yml` 增加 `npx playwright install chromium` 并改为执行 `npm run verify`。
6. 验证结果：
   - `npm run test:browser` 通过，`2/2` 通过。
   - `npm run verify` 通过。
