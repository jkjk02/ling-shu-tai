## 实施步骤

1. 先落浏览器测试专用隔离后端启动脚本。
2. 再把 Playwright `webServer` 接线切到隔离入口。
3. 然后新增 managed MCP 浏览器真实提交用例。
4. 最后执行 `npm run test:browser` 与 `npm run verify`，回填文档。
